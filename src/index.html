椰子云笔记 （CoconutNote）

项目预览链接
  • 技术栈： Vue 2、JavaScript、AJAX、Vuex 。
  • 项目介绍：该项目是一款基于 Vue 2、JavaScript 构建的轻量级云笔记 SFC 项目。适配 PC 端网页，包含欢迎、登录（注册）、笔记本列表、笔记详情、回收站等多个页面，主要组件有侧边栏导航等。 在注册（登录）后，可根据不同的类型创建多个笔记本，笔记本内可创建多个笔记，用户可随时记录、编辑、删除笔记。无论是工作安排、学习笔记、随想，即开即用，方便你的工作和生活。 使用 CodeMirror 组件，支持 Markdown 语法，点击预览可显示笔记实际效果。用户账号信息和所有笔记，均存储在云端服务器，被删除笔记将暂存在回收站中，可恢复或彻底删除。
  • 项目经验：基于 Vue 2、JavaScript、vue-cli 脚手架开发，原生支持较好，节省配置时间。 对约定的 HTTP 请求进行封装，提高了 AJAX 的编写效率，在用户数据同步的 HTTP 请求中使用 Promise 语句，链式调用避免了回调地狱，项目开发更加便利。 使用 Vuex 管理用户数据状态，大幅减少了组件通信的繁杂，拥有 state 、mutations 、modules、actions、getters ，解决了多组之间状态共享问题。 使用路由定位被编辑的笔记，用 query 方法进行请求，并监听 query id 的变化，实现数据的更新和页面切换。 Vue 2 框架使用原生 JavaScript 开发体验良好，但类型推断能力弱，注意厘清数据的形式。 使用 Chrome 调试时安装 Vue.js 的 devtools 插件，可以很简便地对数据结构进行解析和调试，能直观的查看各组件的 data、computed ,以及 Vuex 管理的各项数据状态。

有余记账 （surplus-app）

项目预览二维码

项目预览链接
  • 技术栈： Vue 3、Vite 3、TSX 、Pinia 。
  • 项目介绍：该项目是一款基于 Vue 3、TSX 构建的在线记账 SFC 单页面应用。 可以进行日常生活中各项收支的记录和分类，支持添加标签 Emoji 表情和编辑、金额、备注，查看历史开支信息。 具备信息可视化查看功能，用户各项数据，如标签、备注、收支金额、记录时间等，均存储在云端服务器。 可适配移动端、平板、 PC 端，特别对移动端进行优化，在手机上使用体验较好。 主要页面有:登录（注册）、初次使用欢迎页、记账页面、统计图表、记账页面和标签选择页面，点击左上角选项图标会显示侧边导航栏，可以直接点击跳转对应的页面，顶部可进行登录和退出账号的操作。 所有收支明细可按本月、上月、今年和自定义时间的统计模式查看，统计图表页面中也可以按本月、上月、自定义时间生成对应的可视化折线图、饼图、条形图。
  • 项目经验: 本项目用 TSX 语法代替 template 模板，同时结合 CSS Modules 代替 style 标签。 使用 Vue Router 通过路由守卫实现鉴权，进而引导用户登录，自制 Vite 插件实现 SVG Sprites ，同时还实现了广告页的手势滑动效果。 自制表单组件，包含输入框文本、验证码、日期、下拉列表、 Emoji 表情等，可扩展性强。 使用 Axios 处理网络请求，采用 JWT 进行用户认证，进行了二次封装，包含错误处理、显示加载中提示登录等功能，据此自制了纯前端 Mock 系统，使用 faker 构造数据。 全局状态管理使用 Pinia ，避免封装的 store 导致的跨请求状态污染，记录用户的登录状态和相关数据，进而熟悉了 Pinia 的 ID 持久化等知识。 使用 ECharts 实现数据可视化，包含条形图、折线图、饼图等，了解如何构造数据、修改相应配置项。 通过 dynamic import 、 Rollup chunk 、条件编译等方法优化项目打包体积。 项目尽量减少了对第三方组件库的依赖，只使用了 Vant 组件，提高了学习和封装能力。
